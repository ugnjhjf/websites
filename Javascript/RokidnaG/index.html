    Kurumi Server

    Kurumi Server Project is a integrated project about software and hardware. This project started at 2018. The server is self-hosted on 2U DELL R720 Rack Server.Running Vmware ESXI to provide virtual machine (VM) platform for studying, testing new technology and technic.

    Here are projects and technology have been implemented on the server and deployed.

    Software

      Backend technology:
    1. Nginx - Reverse Proxy, HTTPS, Webbsite deploy
    2. MySQL & MongoDB - Database
    3. Vmware ESXI - Virtual Machine Management Platform
    4. Dell iDRAC - Rack server life cycle management system
    5. DDNSGO - ddns service for dynamic IP
    6. Dell Raid Card - Raid 10

      Service deploy:
    1. Alist - Private cloud storage & NAS
    2. Jellyfin - Online video streaming platform
    3. AutoBangumi - Work with Jellyfin, to provide media information
    4. Minecraft Server

    2. DDR3 ECC RAM 1333MHZ 64G
    3. SFP+ 10Gbps *2 network card
    4. Nvidia Geforce GTX 750Ti
    5. 8TB HDD * 4 - Raid 10
      Youcan 3-level switch (48 ports, 4 SFP+ fibre ports)
